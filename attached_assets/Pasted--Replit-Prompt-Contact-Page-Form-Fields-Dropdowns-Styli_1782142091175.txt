## SECTION 4 — FIELD STYLING

Match the clean underline style shown in Images 2 & 3:

```css
/* Field label */
.form-label {
  font-size: 11px;
  letter-spacing: 0.12em;
  color: #888880;
  text-transform: uppercase;
  font-weight: 400;
  margin-bottom: 8px;
}

/* Text inputs and selects — underline only, no box */
.form-input,
.form-select {
  border: none;
  border-bottom: 1px solid #C8C0B0;
  border-radius: 0;
  background: transparent;
  padding: 8px 0;
  font-size: 15px;
  color: #2C2C2A;
  width: 100%;
  outline: none;
}

.form-input:focus,
.form-select:focus {
  border-bottom-color: #C9A96E; /* gold on focus */
}

/* Placeholder color */
.form-input::placeholder {
  color: #AAAAAA;
}

/* Textarea */
.form-textarea {
  border: 1px solid #C8C0B0;
  border-radius: 2px;
  padding: 12px;
  font-size: 15px;
  color: #2C2C2A;
  width: 100%;
  min-height: 120px;
  outline: none;
  background: transparent;
  resize: vertical;
}

.form-textarea:focus {
  border-color: #C9A96E;
}

/* Grid layout */
.form-grid {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 32px 40px;
}

.form-field-full {
  grid-column: 1 / -1; /* full width for textarea */
}
```
