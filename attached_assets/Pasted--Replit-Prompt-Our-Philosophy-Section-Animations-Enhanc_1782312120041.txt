## ANIMATION 5 — IMAGE PARALLAX ON SCROLL

Apply a subtle parallax to the bedroom image as user scrolls:

- On scroll, move the image `translateY` at 30% of scroll speed (so it moves slower than the page)
- Range: approximately `-20px` to `+20px`
- Use `requestAnimationFrame` for smooth performance
- Apply transform on the image element, NOT the container

```js
window.addEventListener('scroll', () => {
  const section = document.querySelector('.philosophy-section');
  const img = section.querySelector('.philosophy-image');
  const rect = section.getBoundingClientRect();
  const progress = -rect.top / window.innerHeight;
  img.style.transform = `scale(1) translateY(${progress * 30}px)`;
});
```

Make sure image container has `overflow: hidden` so edges don't show.
